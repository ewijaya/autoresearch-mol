\begin{abstract}
Molecular transformers typically adopt architectures from NLP without evaluating domain fit. We ask whether molecular sequences require different transformer designs, and whether autonomous architecture search adds value beyond hyperparameter (HP) tuning. We deploy an LLM-guided search agent across three sequence domains (SMILES, protein, English text) in a 4-condition design that decomposes architecture versus HP contributions. Across 3{,}106 single-GPU experiments, architecture search value is \emph{domain-dependent}: it contributes 81\% of improvement for NLP ($p = 0.008$), yet is counterproductive for SMILES, where HP tuning alone captures 151\% of improvement ($p = 0.001$). Agent-discovered architectures cluster by domain ($p = 0.004$), yet all 41 innovations transfer across domains with $<$1\% degradation, suggesting search dynamics rather than fundamental domain requirements drive specialization. We provide a practitioner decision framework linking vocabulary size and sequence length to recommended search strategy, and release an open-source framework for autonomous architecture search on arbitrary sequence domains.
\end{abstract}
